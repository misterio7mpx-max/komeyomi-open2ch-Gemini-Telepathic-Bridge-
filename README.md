📖 Komeyomi (Rebuild) 設定・使い方マニュアル
1. 基本的な使い方
コメントの送信: 画面右下の入力欄から行います。

設定の開閉: 入力欄の左側にある「⚙」ボタンをクリックすると、画面中央に設定パネルが開きます。もう一度押すか、パネル右下の「閉じる」を押すと閉じます。

画面の配置: 画面の右端に「サイドバー」、そのすぐ左下に「標準の入力枠（komediv）」と「ツールバー（⚙ 枠 送信）」が整列して配置されます。

2. 設定パネルの解説（各タブ）
📏 【サイズ】タブ
画面各所の大きさを直感的にスライダーで調整できます。

サイドバー関連: 幅、文字サイズ、枠内の余白、枠同士の隙間を調整できます。

字幕の文字サイズ: 「流れる字幕（テロップ）」と「ユーザー字幕」の文字サイズを変更できます。

💡 自動高さ調整機能: 文字サイズを変更すると、枠の高さも自動でピッタリのサイズに計算されて調整されます。サイト全体がそれに合わせて自動で押し下げられるため、表示が被ることはありません。

🎙️ 【音声】タブ
自分と視聴者で、それぞれ全く別の音声エンジンや声質を設定できます。

エンジンの選択: VOICEVOX、COEIROINK、AssistantSeika、棒読みちゃんから選択可能。

話者の自動取得: VOICEVOXやCOEIROINKが起動している状態で設定画面を開くと、キャラクター名が自動で読み込まれ、数字（ID）ではなく**「名前」で選択**できるようになります。

視聴者のランダム再生: 「+ 話者を追加」ボタンで複数のキャラクターを登録できます。視聴者がコメントすると、その中からランダムな声で読み上げられます。

🎚️ 音声パラメータの超微調整機能:

話速（スピード）、高音（ピッチ）、抑揚（イントネーション）をスライダーで個別に設定できます。

💡 マウスホイール連動: スライダーの上にマウスカーソルを乗せてホイールを回すと、「0.01」単位で精密な微調整が可能です。

※ スクロールエリアの右端に余白（空白）があるため、ページ自体を上下にスクロールしたい場合はそこを使用してください。

🔤 【字幕】タブ
見た目に関する詳細な設定を行います。

個別フォント設定: 「流れる字幕」「ユーザー字幕」「サイドバー」の3箇所で、それぞれ別のフォント（標準、Noto Sans、丸ゴシック、Dela Gothic One、Yusei Magic、教科書体）を設定できます。

ユーザー字幕の配置: 枠内での文字の位置（上・中・下、左・中央）を自由に変更できます。

カラー変更: 各エリアの背景色をカラーピッカーで変更できます。

流れる字幕の速度: テロップが右から左へ流れる秒数を設定します（数字が小さいほど速くなります）。

🔌 【接続】タブ
外部ツールとの連携や、キーボードの挙動を設定します。

↩️ Enterキーの挙動反転:

一番上のチェックボックスをオンにすると、**「Shift+Enterで送信 / Enterで改行」**に挙動が逆転します。長文をじっくり打ち込みたい時におすすめです。

各エンジンのURL: VOICEVOX等の連携先URLです。基本的にはデフォルト（127.0.0.1:ポート番号）のままで動作します。

3. 🎥 OBSで配信画面に取り込む際のコツ
今回のアップデートで、ユーザー字幕枠の左右に「グレーの破線（ガイド線）」が追加されています。これはOBSでのクロップ（切り抜き）を劇的に簡単にするための機能です。

OBSの「ブラウザソース」または「ウィンドウキャプチャ」でサイトを取り込みます。

ユーザー字幕エリアを見ると、文字を入力していなくても左右に「破線」が見えます（この幅が最大20文字分の固定幅です）。

OBS上で Alt キー（Macの場合は Option キー）を押しながらソースの端をドラッグし、**この破線の位置に合わせて左右をクロップ（切り抜き）**してください。

これで、文字が長くなっても途切れることなく、常に完璧なサイズで配信画面に字幕を配置できます。
