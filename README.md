📌 概要
おーぷん2chのコメントを「棒読みちゃん」「VOICEVOX」「COEIROINK」で自動読み上げするTampermonkeyスクリプトです。画面左下の「⚙️」ボタンから、ブラウザ上で直接設定や音声の微調整が可能です。

⚠️ 免責事項
本スクリプトの使用により生じたいかなるトラブルや不利益についても、作成者は一切の責任を負いません。自己責任でご利用ください。

🛠 必要なもの
本スクリプトを動かすために、以下の2つをご準備ください。

Tampermonkey（ブラウザの拡張機能）

音声読み上げソフト（※使いたいものを事前に起動しておきます）

棒読みちゃん（設定の「HTTP連携」を有効にし、ポートを 50080 に設定）

VOICEVOX * COEIROINK

🚀 使い方
使いたい音声ソフトを起動した状態で、おーぷん2chのスレッドを開きます。

画面左下に表示される「⚙️」ボタンを押し、使用するソフトとキャラクター(話者)を選びます。

スレッドに新しいコメントが投稿されると、自動で読み上げられます。

💡 困ったときのヒント

音が出ない時は？: 音声ソフトが裏で起動しているか確認し、パネル内の「🔄」ボタンを押して話者リストを再取得してください。

テスト再生: パネル内の「▶️」ボタンを押すと、現在の設定でテスト音声が流れます。

微調整: 各スライダーや数値枠の上で「マウスホイール」を回すと、0.01単位で音量や話速を細かく調整できます。
